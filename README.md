# CommonLink

Plateforme de philanthropie transparente où chaque don est certifié on-chain.

## Démarrage

### 1. Base de données
```bash
docker compose up -d
```

### 2. Backend (Spring Boot)
```bash
cd api
./gradlew bootRun
```

### 3. Frontend (Next.js)
```bash
cd app
npm install
npm run dev
```
